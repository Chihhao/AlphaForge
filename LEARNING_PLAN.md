# 🚀 AlphaForge 學習與開發計畫書

本計畫旨在透過開發 **AlphaForge** 平台，同步達成「掌握股票投資」、「精通量化分析」以及「實作 AI Agent」三大目標。

---

## 📅 階段一：建立盤感與指標拆解 (從 ECF 學習)
**目標：** 利用 ECF 工具作為「字典」，學習基本股票術語與 K 線型態。

*   **股市學習**：
    *   每天挑選 1-3 檔股票放入 ECF，觀察其「自動標註」的 K 線型態（如：吞噬、晨星）。
    *   透過 AI 問答，理解指標（MA, KD, RSI, MACD）的數學原理與物理意義。
*   **AlphaForge 開發**：
    *   實作「指標字典頁面」：將 ECF 的靜態指標轉化為 AlphaForge 的 UI 元件。
    *   串接基礎資料源：確保 Python 後端能穩定抓取 Yahoo Finance/TWSE 的歷史數據。
*   **AI Agent 學習**：
    *   學習如何給予 AI Agent (即 Antigravity) 明確的「指令規範」，讓 AI 協助生成指標計算代碼。

## 📅 階段二：公式提煉與自訂化 (打造專屬武器庫)
**目標：** 在 AlphaForge 中實現讓使用者「自定義公式」的功能。

*   **股市學習**：
    *   學習「條件組合」：例如「KD 低檔 + 成交量翻倍」代表什麼意義？
    *   理解「濾網」觀念：如何剔除雜訊。
*   **AlphaForge 開發**：
    *   **Formula Builder**: 開發前端介面，讓使用者能選取指標並設定參數（如：`MA(5) > MA(20)`）。
    *   **Backend Calculator**: 在 Python 端動態解析這些公式，並計算出買賣訊號點。
*   **AI Agent 學習**：
    *   **代碼生成 (Code Generation)**：讓 AI Agent 根據使用者的白話文需求（如：「我想要找最近三天都漲連的股票」）自動轉換為 Python 邏輯。

## 📅 階段三：歷史回測與科學驗證 (真理檢驗)
**目標：** 建立回測引擎，用「數據」告訴你公式是否有用。

*   **股市學習**：
    *   學習「期望值」與「勝率」：理解為什麼高勝率不代表高獲利。
    *   認識「最大回撤 (MDD)」：學習控制風險。
*   **AlphaForge 開發**：
    *   開發 **Backtest Engine**：讓使用者選定一段歷史區間（如過去 5 年），模擬執行該公式。
    *   產出績效報表：包含總回饋率、勝率、賺賠比、資金曲線圖。
*   **AI Agent 學習**：
    *   **數據分析 Agent**：訓練 AI 如何分析回測結果，並主動給出優化建議（例如：「調整停損比例可以讓報酬率提高 10%」）。

## 📅 階段四：自動化通知與 AI Agent 進階化 (實戰自動導航)
**目標：** 系統 24 小時監控盤勢，主動通知買賣訊號。

*   **股市學習**：
    *   實戰心理學：學習在訊號出現時，如何克服恐懼或貪婪去執行。
*   **AlphaForge 開發**：
    *   **Scheduler**: 實作每日盤後自動掃描全台股任務。
    *   **Notification**: 整合 LINENotify 或 Telegram，當公式觸發訊號時主動推播。
*   **AI Agent 學習**：
    *   **自主 Agent (Autonomous Agent)**：實作一個會「自我檢討」的 Agent。它會觀察通知後的股價動向，並在隔日總結：「昨天的買入訊號正確/錯誤，原因可能是...」。

3.  **AlphaForge App**：負責將上述兩者的知識沉澱，轉化為可以重複使用的自動化工具。

這份計畫完成後，您將不只擁有一套能自動選股的系統，更具備了開發「AI 配對量化交易」這類高階應用的實戰能力。
